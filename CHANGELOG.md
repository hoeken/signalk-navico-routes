# v1.0.0

Initial release.

- Discover Navico MFDs (B&G, Simrad, Lowrance) on the local network via GoFree
- Download routes and waypoints from the MFD as USR files and import them into SignalK
- Upload SignalK routes and waypoints to the MFD via manual webapp upload
- Webapp UI with MFD discovery, confirmation dialogs, and export support
- Plugin config panel with periodic polling (default 10 minutes)
